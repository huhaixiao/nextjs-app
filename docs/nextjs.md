# nextjs

- components inside the app folder are ssr
- `'use client';` -> client component include in js bundle
- Server Side Fetching
- Static Rendering
- Dynamic Render
- nextjs caches data by default

```shell
npx create-next-app@latest
```

- Client-side
- Server-side
